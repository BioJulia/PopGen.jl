pairwise_fst(data::PopData; method::String, iterations::Int64)

Calculate pairwise FST between populations in a PopData object. Set iterations to a value greater than 0 to perform a single-tailed permutation test to obtain P-values of statistical significance.

Methods:â€‹

  • "Nei87": Nei (1987) method
  • "WC84" : Weir-Cockerham (1984) method (default) Examples

wc = pairwise_fst(data, method = "WC84")
wc_sig = pairwise_fst(data, iterations = 1000)
